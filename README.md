# Java面试笔记

收集整理GitHub上的Java面试资料，包含Java基础、面向对象、集合框架、多线程、JVM、Spring Boot、数据库、工具经验等内容。

## 项目结构

```
Java-Interview-Notes/
├── README.md                 # 项目说明
├── java-interview-materials.md  # 基础版面试材料
├── java-interview-materials-detailed.md  # 详细版面试材料
├── docs/                     # 文档目录
│   ├── java-basic.md         # Java基础
│   ├── oop.md                # 面向对象
│   ├── collections.md        # 集合框架
│   ├── multithreading.md     # 多线程
│   ├── jvm.md                # JVM
│   ├── spring-boot.md        # Spring Boot
│   ├── database.md           # 数据库
│   └── tools.md              # 工具经验
└── resources/                # 资源目录
    ├── images/               # 图片资源
    └── code/                 # 代码示例
```
